SurfaceToTUIO
-------------
The SurfaceToTUIO program accepts contact events from a MS Surface (or simulator) and translates them into standard TUIO messages.

The SurfaceToTUIO VS-Project Solution needs MS Visual Studio 2008 (C#) and the MS Surface SDK.

References:
-----------
This example is using the OSC.NET OpenSound Control library for C#
along with changes and improvements by the author of reacTIVision, Martin Kaltenbrunner <mkalten@iua.upf.edu>.
http://luvtechno.net/d/1980/02/open_sound_control_for_net_2.html
http://reactivision.sourceforge.net/
